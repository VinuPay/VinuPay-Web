About VinuPay

VinuPay is an e-commerce system and a name service at the same time! It has been written in Solidity++, the smart contract language on the Vite network. It is fully open source on GitHub under MIT. Online stores can use the product to integrate Vita Inu & other Vite tokens as a payment method. Regular users can use it as a name system to simplify payments and remove the need to know the Vite address of the recipient.

VinuPay App

VinuPay App is a web application that allows you to manage your names and invoices! Here are some of its features:

  • Manage your names
  • Mint new names
  • Lookup names
  • Create invoices
  • Pay invoices
  • Lookup invoices

VinuPay.js

VinuPay.js is a JavaScript library that allows you to create and manage names & transactions! Here are some of its features:

  • Mint new names
  • Create an invoices
  • Lookup names
  • Transfer names
  • Retrieve statistical data about the contract, such as the number of registered names and the number of invoices created
  • Pay invoices
